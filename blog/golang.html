Golang: About, Use cases and its philosophy

About

Golang is a high-level open source programming language supported by Google and created by Robert Griesemer, Rob Pike, and Ken Thompson at Google on 2009. It is a statically-typed and a compiled language known for its simplicity and efficiency. As the legend Terry Davis once said, "An idiot admires complexity, whereas a genius admires simplicity." and Go follows the same principle. Golang is well known for its garbage collection preventing memory leaks. It is statically-typed and is cross

Features

  • Statically-typed: Determines type of the variable during compile times and catches type-related errors
  • Concurrency: Supports running multiple processes at the same time, which is useful in building faster and scalable network servers.
  • Cross-platform: It can run on any hardware, Operating Systems and frameworks.
  • Standard Library: It has robust and rich source of standard libraries.

Use Cases

  • Cloud and Network services: Go is often described as Cloud's language due to its heavy influence on Cloud computing and Networks.
  • CLI Tools: There are lots of standard libraries that are used to create CLI tools and interfaces
  • DevOps: Most of the DevOps tools were developed by Go developers and plays a important role in Docker containerization.
  • Web Development: Go can be used to build robust and scalable backend and web applications.
